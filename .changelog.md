## [2.4.0] - 2026-03-15
### Changed
- **Bounce Density Physics**: Rebalanced the bounce-count formula to better reflect low-trajectory clubs.
- **Trajectory Integration**: Added `dynamicLoft` as a variable in determining bounce frequency. A well-struck Driver (10-12 degrees) now consistently produces 4-5 skips on firm fairways.## [2.3.0] - 2026-03-15

### Added
- **Dynamic Bounce Audio:** Replaced the static landing thud with a dynamic bounce sequence.
- **Trajectory-Based Ground Interaction:** The number of bounces, the timing between them, and their volume now scale mathematically with the `rollDistance`. High-loft clubs (Wedges) produce a single heavy thud, while low-loft clubs (Driver, Woods) produce multiple cascading, skipping bounces before the roll out.## [2.1.0] - 2026-03-15

### Added
- **Golf Bag Array:** Introduced 11 selectable clubs (Driver through Sand Wedge), each with specific base distances, lofts, and roll percentages.
- **Club Selection:** Mapped `PageUp` and `PageDown` to cycle through clubs while in State 0 (Idle), complete with `window.announce` feedback.
- **Flight Phase (State 5):** Added a new state that locks out keyboard input while the ball is "in the air."
- **Trajectory & Roll Physics:** Introduced dynamic loft and roll calculations. Fat/Thin hinge misses now directly impact hang time and roll-out distance.
- **Audio Sequencer:** Replaced instant impact reporting with a sequenced audio event: Strike (triangle wave) -> Flight Wind (white noise) -> Landing (sine thud) -> Roll (brown noise rumble) -> Caddie Reveal.
